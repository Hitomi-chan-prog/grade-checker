# グレードチェック

## 概要
生徒の人数、名前、点数を入力すると、合格 / 不合格と評価（A / B / C）を表示するPythonツールです。

## 機能
- 人数を入力して複数人の成績を登録
- 名前と点数を入力して成績判定
- 合格 / 不合格の表示
- A / B / C 評価の表示
- 平均点の計算
- 合格者一覧の表示

## 使用技術
- Python

## 判定ルール
- 90点以上: A
- 80点以上: B
- 80点未満: C

python main.py
